# Project Images

Place your project images in this directory. Reference them in App.js like so:

```jsx
<img src="/projects/image-name.jpg" alt="Project Name" />
```

## Recommended Images:

### Featured Projects:
- `cryptsim.jpg` or `.webp` - CryptSim trading simulator
- `chronicles.jpg` or `.webp` - Chronicles of Elijah RPG game
- `vapour.jpg` or `.webp` - Vapour game distribution platform

### Other Projects:
- `portfolio-v1.jpg` or `.webp` - First portfolio
- `laragigs.jpg` or `.webp` - Laragigs job listing platform
- `wordpress-demo.jpg` or `.webp` - WordPress demo site

## Tips:
- Use `.webp` format for better performance (smaller file size, better quality)
- Recommended size: 400x250px for featured projects, 300x200px for other projects
- Optimize images for web to keep bundle size small
